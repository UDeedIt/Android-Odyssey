# Android Odyssey – Specification (Initial Draft)

## Purpose

Android Odyssey is a didactic Android app that demonstrates modern Android
development practices (Jetpack Compose, Dagger 2, modular architecture,
testing, CI/CD) while acting as a learning companion for developers.

At this stage, the goal of this document is to provide a high-level view of
the intended functionality and technical direction. It will evolve as the
project grows.

## High-level user goals (early draft)

- Browse structured Android learning content and code examples.
- See how a real-world-quality Android app is architected.
- Understand how DI, coroutines, navigation, and testing fit together.

## Non-functional goals

- Clean, well-documented Kotlin code with KDoc.
- Clear modular architecture.
- Strong automated checks (tests, lint, CI).
- Demonstrate professional Git workflow (Git Flow).
