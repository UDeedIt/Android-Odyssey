# Android Odyssey

Android Odyssey is an open-source Android learning guide and demonstration app
built with Jetpack Compose. The project aims to showcase modern Android
architecture, dependency injection, coroutines, testing, and CI/CD in a clear,
well-documented codebase.

> Status: Early setup phase (project structure, DI, and CI).

## Tech stack (planned)

- Kotlin
- Jetpack Compose
- Dagger 2 for dependency injection
- Coroutines / Flow
- Jetpack Navigation
- GitHub Actions for CI
- ...many more

More documentation will be added as the project evolves.
